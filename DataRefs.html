Custom Docking Guidance Systems

The supplied DGSs are not designed to be customisable. However the plugin publishes the following datarefs that you can use to construct your own DGS. The supplied DGSs include a static reduced level-of-detail version for viewing at distances above 500m. You should do the same in your DGS to reduce the load on X-Plane.

marginal.org.uk/dgs/status
0 - "Idle". All other published datarefs will also have value 0.
1 - "Active". A plane is approaching the gate and the plane's doorway location is known.
2 - "Stop". The plane is at the stopping point.
3 - "Stopped". The plane has stopped engines at the stopping point.
4 - "Too far". The plane is beyond the stopping point.
5 - "ID fail". The plane approaching the gate does not have a valid doorway location.
 
marginal.org.uk/dgs/track
0 - "Idle/Stop/Too far". Either no plane is approaching the gate or the plane is at or beyond the stopping point (status=0, 2, 3 or 4).
1 - "Active". A plane is approaching the gate and the plane's doorway location is known (status=1).
2 - "Tracking". The plane is >6m and <≈60m from the stopping point (status=1).
3 - "Tracking 6m". The plane is <6m from the stopping point (status=1).
 
marginal.org.uk/dgs/lr
0 - "Idle/Active" (status=0 or track=1).
0 - "Tracking on target" (track=2 or 3).
1 - "Tracking left". The plane is to the left of the stopping point (track=2 or 3).
2 - "Tracking right". The plane is to the right of the stopping point (track=2 or 3).
3 - "Stop/Too far/ID fail". The plane is at or beyond the stopping point (track=0).
 
marginal.org.uk/dgs/azimuth
0 if track=0 or 1.
Distance (in increments of 0.5m) that the plane is to the right of the stopping point if track=2 or 3. Negative when the plane is to the left of the stopping point. Has value -4 or +4 if the plane is >4m to the left or right of the stopping point.
 
marginal.org.uk/dgs/distance
0 if track=0 or 1.
Distance (in increments of 0.5m) of the plane from the stopping point if track=2 or 3. Has value 12 if the plane is >12m from the stopping point.
 
marginal.org.uk/dgs/distance2
0 if track=0 or 1.
Distance-6m (in increments of 0.5m) of the plane from the stopping point if track=2. Has value 6 if the plane is >12m from the stopping point.
Distance (in increments of 0.5m) of the plane from the stopping point if track=3.
 
marginal.org.uk/dgs/icao
Array of 4 integers spelling out the ASCII code of the plane's ICAO model designation:
0 if status≠1
space, ‘0’-‘9’ or ‘A’-‘Z’ if status=1
